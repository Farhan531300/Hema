About This Project

Economic freedom is a fundamental driver of prosperity. Understanding global economic patterns through data visualization can help improve awareness, support policymaking, and empower individuals with knowledge.

This project uses Tableau dashboards and stories to analyze key economic indicators such as inflation rates, unemployment figures, GDP growth, and country rankings based on the Economic Freedom Index.

By combining data analytics with visualization, this project aims to make complex economic insights more accessible to students, researchers, economists, and policymakers worldwide.
